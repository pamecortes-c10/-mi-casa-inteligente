# 🏠 Mi Casa Inteligente

PWA para control del hogar — Luces, Clima y Spotify.

## Deploy en Render (gratis)

1. Sube esta carpeta a GitHub
2. Ve a render.com → New → Web Service
3. Conecta tu repositorio
4. Render detecta render.yaml automáticamente
5. Clic en Deploy

## Variables de entorno requeridas

| Variable | Valor |
|----------|-------|
| SP_CLIENT_ID | Tu Spotify Client ID |
| SP_CLIENT_SECRET | Tu Spotify Client Secret |
| OWM_KEY | Tu OpenWeatherMap API Key |

## Instalar como app

- **Android**: Chrome → Menú ⋮ → Agregar a pantalla de inicio
- **iPhone**: Safari → Compartir → En pantalla de inicio  
- **PC**: Chrome → ícono ⊕ en la barra de URL
